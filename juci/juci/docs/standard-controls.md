JUCI STANDARD CONTROLS
======================

JUCI comes with a library of already written standard controls that you can use
to create pages and arrange your layout. These can be used by simply using their directives in your html.

JUCI CONTROLS
=============

* [juci-select](juci-select.html) a generic dropdown select box. 
* [juci-switch](juci-switch.html) a switch control 
* [juci-list-editor](juci-list-editor.html) a generic control for editing lists of complex items.
